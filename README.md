# Password-Generator

## Purpose
A website which will help to create a strong password that provides greater security.

## Functions
When we click on generate password button then we will bw asked what length should be the password and it must be between 8 to 128, after that we will be asked if we want upper case characters, lower case characters, numbers and special chracters in our password.One must choose atleast one chracter type in order to make a random password.
